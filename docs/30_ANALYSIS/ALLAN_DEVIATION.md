# Allan Deviation

Allan deviation and related stability metrics are central OTIS analysis tools.

OTIS should eventually support:
- overlapping Allan deviation;
- TDEV;
- MTIE;
- environmental correlation;
- phase residual analysis.

Analysis tooling should preserve:
- reproducibility;
- explicit window definitions;
- explicit preprocessing assumptions.

Derived analysis must always identify:
- source datasets;
- transforms;
- filtering methodology.
