# H0 PPS + TCXO Real Run Template

Template for the first combined H0 hardware run. It is intentionally empty
except for CSV headers and must not be treated as captured hardware data.

Expected wiring is GPS PPS on `D14` / `CH1` and the conditioned/divided TCXO
observation path on `D8` / `GPIO20` / `GPIN0` / `CH2`.
