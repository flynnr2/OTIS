# H1 Open-Loop Run Notes

## Purpose

Capture OCXO free-run behavior with no automatic steering.

## Hardware Setup

Record oscillator identity, pinout source, power state, DAC/control input state, output conditioning, reference source, and RP2040 observation pin.

## Safety Limits

Record the verified control-input and DAC limits even if the DAC is disconnected during this run.

## Capture Command

Record the exact host command used for capture.

## Observations

Summarize warm-up duration before capture, count stability, reference availability, frequency offset, and qualitative noise or dropout behavior.

## Anomalies

Record dropped records, conditioning failures, missing PPS, output interruptions, or host-side interruptions.

## Follow-up

List actions needed before manual DAC sweep or settling runs.
